发表于 2017-02-22 | 分类于 PHP |

申明数组 array()
键可以由数值或者字符串组成
键是数值的数组称为索引数组
键是字符串的称为关联数组
数组的值获取通过 [] 内部是索引

1.创建方式

array()创建

直接赋值

range()创建

2.数组长度 count();

<?php
unset($letter[i]); // 删除索引值为i的值

unset($letter[5]);   // 删除索引值为i的值
echo $letter[5];   // 索引值5被删除了
// Notice: Undefined offset: 5 in xx .php on line x

?>

<?php
print_r($letter);
?>

<?php
in_array(“b”,$letter) // 判断$letter数组中是否存在b
?>

<?php
for($i = 0; $i < count($letter); $i++){
echo $letter[$i];
// ABCDEGHIJKLMNOPQRSTUVWXYZ[]^_`
}
?>

<?php
foreach($letter as $key => $value){
echo $key.”的值是”.$value.”

“;
}

foreach($letter as $value){
        echo $value;
        // ABCDEGHIJKLMNOPQRSTUVWXYZ[\]^_`
}

?>

<?php
$fruit = array(“banana”,”orange”,”apple”);
rsort($fruit);
print_r($fruit); // Array ( [0] => orange [1] => banana [2] => apple )
?>

<?php
$fruit = array(“b”=>”orange”,”a”=>”banana”,”c”=>”apple”);
ksort($fruit);
print_r($fruit); // Array ( [a] => banana [b] => orange [c] => apple )
?>

<?php
asort($fruit);
print_r($fruit); // Array ( [c] => apple [a] => banana [b] => orange )
?>

<?php
$people = array(
array(“name”=>”zhangsan”,”sex”=>”man”),
array(“name”=>”lisi”,”sex”=>”woman”)
);

print_r($people);
// Array ( [0] => Array ( [name] => zhangsan [sex] => man ) [1] => Array ( [name] => lisi [sex] => woman ) )

?>

<?php
for($i = 0; $i < count($people); $i++){
foreach($people[$i] as $key => $value){
echo $people[$i][$key].”

“;
}
}

foreach($people as $key => $newArr){
    var_dump($newArr);
    foreach($newArr as $key => $value){
        echo $key."的值是".$value;
        echo "<hr/>";
    }
}

for($i = 0; $i < count($people); $i++){
    var_dump($people[$i]);
    foreach($people[$i] as $key => $value){
        echo $value."<hr/>";
    }
}

?>

<?php
$arr = array(1,2,3,4,5);
$str = implode(“.”,$arr);
echo $str; // 1.2.3.4.5
?>

<?php
$str = “a,b,c,d,e”;
$arr = explode(“,”,$str);
print_r($arr); // Array ( [0] => a [1] => b [2] => c [3] => d [4] => e )
?>
