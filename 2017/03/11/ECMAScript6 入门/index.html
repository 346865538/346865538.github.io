ECMAScript6 入门

ECMAScript 6.0（以下简称 ES6）是 JavaScript 语言的下一代标准，已经在2015年6月正式发布了。它的目标，是使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。

Babel ES6 在线转换工具

for-of 循环

迭代器

ES6 全新的迭代器概念 他允许我们在语言层面上定义一个序列

generator 函数 (生成器)

普通的JS函数被调用时，他们立刻运行，直到遇到return或抛出异常时才会停下来
而现在当你调用一个生成器时，他不会立即执行，而是返回一个已暂停的生成器对象(test)
每当你调用生成器对象的.next()方法时，函数调用将其自身解冻并一直运行到下一个yield表达式，再次暂停。

` 模板字符串

ES6中一种新的字面量语法 模板字符串
可以使用反撇号字符 ` 代替普通字符串的引号 ‘ 或 “

小栗子：

模板字符串也可以多行书写
模板字符串中所有的内容都会原样输出在生成的字符串中

不定参数

newArr前的省略号表明它是一个不定参数 所有传递进来的参数都被放到一个数组中并赋值给变量newArr

所有函数参数中，只有最后一个才可以被标记为不定参数，他之前的所有参数都正常填充，多余的参数被赋值给不定参数。如果没有多余的参数，不定参数就是一个空数组

默认参数

Destructuring 解构

解构赋值允许你使用类似数组或对象字面量的语法将数组和对象的属性赋给各种变量

数组的解构

var [a, …TestArr] = [1, 2, 3, 4];
console.log(TestArr); // [2, 3, 4]

var x = { name: “zhangsan” };
var y = { name: “lisi” };
var { name: nameA } = x;
var { name: nameB } = y;
console.log(nameA); // 张三
console.log(nameB); // 李四

var { a, b } = { a: “zhangsan”, b: “lisi” };
console.log(a); // “zhangsan”
console.log(b); // “lisi”

// ES5
var Test = function(num){
console.log(num)
}

// ES6
var Test = (num)=> console.log(num)

// ES5
var Test = function(){
a();
b();
}

// ES6
var Test = ()=> {
a();
b();
}

var a = Symbol();
console.log(a); // Symbol()
console.log(typeof(a)) // symbol

var name = Symbol();
var man = new Object();

man[name] = “vipcw”;

console.log(man[name]); // vipcw
